# brainmaker

## what's brainmaker ?
brainmaker is a library which can help you to write brainfuck programs.

## show me an example
```
s4r@crackme ~/braincooler$ cat helloname.py
import brainmaker

bf = ''
bf += brainmaker.writeString("enter your name:\n")
bf += brainmaker.gotoPtr(0, 5)
bf += brainmaker.getLine()
bf += brainmaker.gotoPtr(5, 0)
bf += brainmaker.writeString("hello ")
bf += brainmaker.gotoPtr(0, 5)
bf += brainmaker.writeCells()

print(bf)

s4r@crackme ~/braincooler$ python3 helloname.py > helloname.bf

s4r@crackme ~/braincooler$ cat helloname.bf
[-]+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++.+++++++++.++++++.---------------.+++++++++++++.----------------------------------------------------------------------------------.+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++.----------.++++++.---.----------------------------------------------------------------------------------.++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++.-------------.++++++++++++.--------.-------------------------------------------.------------------------------------------------.>>>>>+[->,----------]<[+++++++++++<]<<<<<[-]++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++.---.+++++++..+++.-------------------------------------------------------------------------------.>>>>>>[.>]<[<]

s4r@crackme ~/braincooler$ python3 braincooler.py helloname.bf

s4r@crackme ~/braincooler$ ./a.out
enter your name:
stanislas
hello stanislas
```

# brainbugger

## what's brainbugger ?
brainbugger is a simple debugger written in python3 with curses, which help you to debug your brainfuck programs.

## usage
`python3 brainbugger.py examples/helloworld.bf`
